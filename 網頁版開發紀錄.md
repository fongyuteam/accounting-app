# 帳務管理系統 — 網頁版開發紀錄

## 目前狀態
原本的 Electron 桌面版已改成 Express 網頁版，可在內網多台電腦同時使用。

---

## 網頁版架構

- **伺服器**：Windows 電腦跑 `server.js`（Express + SQLite）
- **前端**：瀏覽器開啟，HTML/CSS/JS 幾乎不變
- **啟動方式**：點兩下 `start.bat`
- **Port**：3000

### 新增檔案
```
server.js          ← Express 伺服器（替換 Electron main.js）
google-auth-web.js ← Google OAuth（不依賴 Electron）
start.bat          ← Windows 一鍵啟動腳本
```

### 修改檔案
```
src/renderer.js    ← 頂部加入 window.api（fetch 實作，替換 Electron preload）
src/index.html     ← 底部加入隱藏檔案選擇器
package.json       ← 加入 express 依賴、start:web 指令
```

---

## 使用方式

### Windows（伺服器）
```
點兩下 start.bat
瀏覽器開啟 http://localhost:3000
```

### Mac（客戶端）
```
瀏覽器開啟 http://192.168.1.113:3000
（IP 依實際 Windows 電腦 IP 為準，每次啟動時 start.bat 會顯示）
```

### 指令啟動
```bash
cd "C:\Users\hp\OneDrive\桌面\匯入資料\accounting-app-v2"
node server.js
```

---

## 網頁版已知限制

1. **拍照功能**：Mac 用 http 區網連線時鏡頭被瀏覽器擋（Safari 較寬鬆）
2. **伺服器要開著**：Windows 那台要一直開著才能用
3. **目前只支援內網**：外網需額外設定
4. **開啟資料夾**：只對 Windows 本機有效
