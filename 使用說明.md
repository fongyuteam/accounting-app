# 帳務管理系統 v2 — 使用說明

---

## 啟動程式

### Windows（伺服器）
點兩下 `start.bat`，黑色視窗出現後保持開著不要關。
瀏覽器會自動開啟，或手動輸入：
```
http://localhost:3000
```

### Mac（連線使用）
確認 Windows 電腦已啟動後，在 Mac 的瀏覽器輸入：
```
http://192.168.1.113:3000
```
（IP 依實際情況為準，啟動時黑色視窗會顯示）

> ⚠️ Windows 那台電腦要保持開著，關掉大家就無法使用

## 功能說明

### 一、拍照辨識（AI）
用 AI 自動辨識單據並新增到帳務。

**Windows：**
- 選擇圖片上傳，AI 自動辨識

**Mac：**
- 點「開啟鏡頭拍照」直接拍照
- 或選擇圖片檔案上傳

**使用步驟：**
1. 上傳或拍攝單據圖片
2. AI 辨識完成後確認廠商、金額、日期等資訊
3. 類別可修改，選「其他（自訂）」可輸入自訂類別名稱
4. 點「確認並新增」完成

> 需要先設定 Anthropic API Key 才能使用 AI 功能
> 取得位置：console.anthropic.com

---

### 二、Excel 匯入
大量匯入帳務資料。

**方式一：固定格式匯入（不需要 API Key）**
1. 點「下載範本」取得 Excel 範本
2. 填寫入帳、出帳、應收帳款三個工作表
3. 點「選擇 Excel 匯入」上傳填好的檔案

**方式二：AI 智慧匯入（需要 API Key）**
1. 點「選擇 Excel 檔案」上傳任意格式的 Excel
2. 點「AI 分析」讓 AI 自動判斷每筆資料
3. 確認預覽後點「確認匯入」

---

### 三、客戶付款追蹤
管理應收帳款，追蹤哪些客戶還未付款。

- **手動新增**：填寫客戶、金額、應付日期等
- **CSV 匯入**：點「選擇 CSV 匯入」批量匯入
- **Google Drive 同步**：自動從 Google Drive 抓取請款單（見下方說明）
- **確認收款**：點「✓ 確認收款」後自動同步到入帳管理，來源標示「付款確認」
- 逾期未付款的會顯示 ⚠ 警示

---

### 四、入帳管理
記錄所有收入。

- 手動填寫日期、客戶、款項名稱、金額、類別
- 類別選「其他（自訂）」可輸入自訂名稱
- 來源標示：手動 / AI辨識 / Excel匯入 / 付款確認

---

### 五、出帳管理
記錄所有支出。

- 手動填寫日期、廠商、支出名稱、金額、類別
- 類別選「其他（自訂）」可輸入自訂名稱
- 每筆都有「編輯」按鈕可修改（包含自訂類別）
- 來源標示：手動 / AI辨識 / Excel匯入

---

### 六、所有紀錄
整合入帳和出帳的總覽。

- 可依類型篩選（全部 / 入帳 / 出帳）
- 可依月份篩選
- 點「匯出 CSV」下載所有紀錄

---

### 七、Google Drive 同步
自動從 Google Drive 的「豐裕請款單」資料夾抓取請款單。

**第一次使用：**
1. 確認 `credentials.json` 放在程式資料夾內
2. 點「登入 Google 帳號」，瀏覽器開啟後完成授權

**每次同步：**
1. 選擇年份（預設 115 年）
2. 勾選要抓取的月份
3. 點「開始抓取」
4. 預覽確認後點「確認匯入」

> 已存在的資料會自動取消勾選並標示「已存在」，不會重複新增

**抓取規則：**
- 目標資料夾：「豐裕請款單」
- 跳過：「已結案」資料夾、請款單總表、倉儲收入報表等
- 金額欄位：優先抓「應付總金額」
- 客戶名稱欄位：從「客戶名稱：」讀取

---

### 八、客戶名單
管理客戶和廠商資料。

- 分「入帳客戶」和「出帳廠商」兩類，完全分開
- 新增入帳/出帳時會自動加入對應名單
- Google Drive 同步時新客戶自動新增到入帳客戶
- 可手動編輯、新增、刪除
- 支援 CSV 匯入

## 類別說明

**入帳類別：** 服務費、產品銷售、顧問費、維護費、其他（自訂）

**出帳類別：** 水電、租金、薪資、設備、軟體訂閱、行銷、差旅、其他（自訂）

> 選「其他（自訂）」後下方會出現輸入框，可填入任意類別名稱（例：保險費、清潔費、油料費）

## 資料儲存位置

```
accounting-app-v2/
  data/
    accounting.db       ← 所有帳務資料（重要！請定期備份）
    google_token.json   ← Google 授權憑證
  credentials.json      ← Google API 憑證
```

> 更新程式時，`data/` 資料夾保留不動，帳務資料就不會消失

## 常見問題

**Q：Mac 鏡頭無法使用？**
A：確認 Safari 或 Chrome 有允許相機權限。系統設定 → 隱私權與安全性 → 相機，確認瀏覽器已勾選。

**Q：Google Drive 抓不到某個廠商？**
A：確認請款單的試算表內有「客戶名稱：XXX」這個欄位，格式必須完全符合。

**Q：AI 辨識費用？**
A：拍照辨識一張約 NT$0.1～0.2，Google Drive 請款單分析一張約 NT$0.02，一個月大概 NT$5～20。

**Q：程式更新後需要重新安裝嗎？**
A：不需要，直接重新點 `start.bat` 就好。只有 `package.json` 套件有變動時才需要跑 `npm install`（會另行提醒）。
